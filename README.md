# IT3040 Assignment 1 - Automated Testing Solution

## Project Overview
Complete automated testing solution for Singlish to Sinhala translation system. Includes:
- 24 positive test cases
- 10 negative test cases
- 2 UI test cases
- Automated execution with Playwright
- Excel-based test management
- Comprehensive reporting

## Quick Start

### Prerequisites
- Node.js 16+
- Git

### Installation
```bash
# Clone repository
git clone https://github.com/Dineth111/Singlish-Sinhala-Automated-Translator-Testing.git
cd Singlish-Sinhala-Automated-Translator-Testing

# Run setup script (if available)
npm run setup

# Or install manually
npm install
npx playwright install
